### Desc
- 태그는 &lt;ev-icon&gt;(이하 <아이콘>)으로 정의

```
<ev-icon icon="아이콘명" />
```

### Props

|    이름     |   디폴트   |  타입   |          설명            |                    종류                           |
|------------ |-----------|---------|-------------------------|---------------------------------------------------|
| icon     | ''      | String | EVUI에서 제공하는 아이콘 클래스명 | |
| size     | ''      | String | 아이콘 | s, m, l |

### Event

 | 이름 | 파라미터 | 설명 |
 | ---- | ------- | ---- |
 | click | event | 클릭 이벤트 감지 |
 | dblClick | event | 더블 클릭 이벤트 감지 |
 | contextmenu | event | 컨텍스트메뉴(우클릭) 이벤트 감지 |
